# SkillMap — Smart Skill Gap Analyzer
**No API key required. 100% free.**

## Setup (3 steps)

### 1. Install Flask
```bash
pip install flask
```

### 2. Run the app
```bash
cd skill_gap_analyzer
python app.py
```

### 3. Open browser
Visit: **http://localhost:5000**

---

## Default Admin Account
- Email: `admin@skillgap.com`
- Password: `admin123`

## Features
- User registration & login (hashed passwords)
- Paste resume text or upload PDF/TXT
- Smart keyword + alias skill extraction (no AI needed)
- Match % with visual ring chart
- Matched vs missing skills breakdown
- Prioritized learning roadmap with free resources
- Analysis history & detail view
- Admin panel to manage roles and users
- 8 pre-loaded job roles
